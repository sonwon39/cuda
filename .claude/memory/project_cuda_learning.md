이 워크스페이스(`c:\Users\son97\cuda`)의 목적:

1. **학습 단계** — CUDA 프로그래밍 기초/심화 개념 학습 (커널, 메모리 모델, 스레드/블록/그리드, 동기화, 스트림, 공유 메모리, warp 등)
2. **실습 단계** — 학습한 개념을 코드로 직접 작성하고 실행/벤치마크
3. **최종 목표** — CUDA를 활용한 앱(애플리케이션) 1개 완성

**Why:** 사용자가 명시적으로 밝힌 이 공간의 장기 목표. 단발성 작업이 아니라 점진적으로 쌓아가는 학습-실습-제작 흐름.

**How to apply:**
- 코드 작성 시 학습용임을 고려해 의도가 드러나는 주석/네이밍을 적절히 사용 (단, 과도한 설명 주석은 지양)
- 새 개념을 도입할 때는 왜 그 기법을 쓰는지 짧게 짚어주면 학습에 도움
- 최종 앱의 방향성이 정해지면 이 메모리를 업데이트하여 구체적인 도메인/목표를 반영
- 빌드 도구체인(nvcc, CMake 등), GPU 사양 등 환경 정보가 확인되면 별도 메모리로 기록
